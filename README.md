# 🚀 C# Fundamentals - Zero to Engineer

Repositório dedicado ao domínio da base técnica em C# e lógica de programação.

## 🛠️ Tecnologias e Conceitos
- **Linguagem:** C# (.NET 8)
- **Fundamentos:** POO (Classes/Objetos), Listas Dinâmicas, Loops (`for`/`foreach`).
- **Lógica:** Acumuladores, Algoritmos de busca (Maior/Menor) e Filtros de dados.

## 📁 Exercícios Práticos
1. **Sistema de Notas:** Média e performance escolar.
2. **Gestão de Estoque:** Controle de inventário e valores.
3. **Gestão de Equipe:** Folha de pagamento e filtros salariais.
4. **Controle de Frota:** Gestão de quilometragem e seminovos.
5. **Cine-Manager:** Duração de títulos e categorias épicas.
